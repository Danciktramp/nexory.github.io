# nexory.github.io
Fashion Store
Clothes & Accessories Store Website
Overview

This is a fully responsive e-commerce website for a clothes and accessories store. The site showcases products such as bags, belts, and apparel, allowing users to browse items with detailed descriptions and multiple images. The website is built with HTML, CSS, and JavaScript and uses JSON to manage product data.

Features

Responsive design for mobile, tablet, and desktop screens.

Product listing with images, categories, and detailed descriptions.

Multiple images per product displayed in a modal on click.

Easy to update products via JSON data file.

Clean and modern user interface.

Simple overlay/modal for product details.

Technologies Used

HTML5 – semantic structure of the site.

CSS3 – styling, responsive layout, Flexbox/Grid.

JavaScript – dynamic content rendering and interactivity.

JSON – product data management (images, description, categories).
